# Phase 4 Summary: Decisions, Risks, and Approvals

## ما تم إنجازه

تم تجهيز الأساس الأولي لمراكز التحكم الثلاثة:

- سجل المخاطر.
- مركز القرارات.
- مركز الاعتمادات.

تم استخدام `data/daily_control_queue.csv` كمصدر لتوليد سجلات أولية قابلة للمراجعة.

## الملفات الناتجة

- `DECISIONS_RISKS_APPROVALS.md`
  - يشرح منطق Phase 4 والمراكز الثلاثة.

- `DECISION_RISK_APPROVAL_SPEC.md`
  - يحدد قواعد توليد المخاطر والقرارات والاعتمادات.

- `database/phase4_decisions_risks_approvals.sql`
  - views وفهارس داعمة لمراكز التحكم.

- `scripts/build_control_registers.py`
  - سكربت توليد السجلات الأولية.

- `data/risks_register.csv`
  - سجل المخاطر الأولي.

- `data/decisions_register.csv`
  - سجل القرارات الأولي.

- `data/approvals_register.csv`
  - سجل الاعتمادات الأولي.

- `data/control_registers_report.json`
  - تقرير ملخص للسجلات.

## ملاحظات جودة

السجلات الناتجة أولية وتحتاج مراجعة PMO. بعض الكلمات قد تحتاج تنقية سياقية، مثل كلمة `عقد` التي قد تعني عقد قانوني أو عقد اجتماع.

لذلك لا تعتبر هذه السجلات معتمدة تلقائياً، بل تعتبر:

- Draft suggestions
- Submitted suggestions للحرج فقط
- Input لمركز التحكم

- لدينا قائمة مهام تحتاج تحديث.
