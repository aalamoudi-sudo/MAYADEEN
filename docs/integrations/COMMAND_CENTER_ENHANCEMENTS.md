# إضافات غرفة القيادة

تمت إضافة سبع قدرات تشغيلية رئيسية للداشبورد، مع مركز ملفات كتعزيز إضافي:

1. مركز التصعيد التنفيذي.
2. مركز الاجتماعات والمحاضر.
3. مساعد PMO اليومي.
4. مؤشر صحة المشروع.
5. سجل الالتزامات.
6. لوحة المدير العام.
7. نظام التذكير الذكي.
إضافة داعمة: مركز الملفات والنسخ.

## الشيتات الجديدة

ينشئ Apps Script الشيتات التالية تلقائياً:

- `Meetings Register`
- `Commitments Log`
- `File Control Register`

## طريقة العمل

- التصعيد التنفيذي وصحة المشروع ولوحة المدير العام والتذكيرات تقرأ من البيانات الحالية: المهام، القرارات، الاعتمادات، وتكليفات PMO.
- الاجتماعات والمحاضر تحفظ في `Meetings Register`.
- الالتزامات تحفظ في `Commitments Log`.
- مركز الملفات يقرأ من `File Control Register`.

## أفضل تشغيل يومي

1. افتح `لوحة المدير العام` صباحاً لمعرفة صورة المشروع.
2. افتح `مركز التصعيد التنفيذي` لمعرفة ما يحتاج تدخل.
3. استخدم `مساعد PMO` لصياغة رسائل اليوم.
4. بعد الاجتماعات، سجل المحضر في `الاجتماعات والمحاضر`.
5. راجع `التذكير الذكي` قبل نهاية اليوم.
