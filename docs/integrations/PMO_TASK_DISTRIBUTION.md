# مركز توزيع المهام من PMO

## الهدف

مركز توزيع المهام يسمح للـ PMO بتكليف الأشخاص مباشرة من الداشبورد، مع:

- تسجيل التكليف في شيت مستقل.
- إرسال إيميل لصاحب المهمة.
- إضافة نسخة للـ PMO عند ضبط `PMO_EMAIL`.
- إرسال إشعار Slack لقناة المشروع.
- ظهور التكليفات العاجلة والمتأخرة في مركز الأولويات.

## اسم الشيت

ينشئ Apps Script شيت مستقل باسم:

`PMO Task Distribution`

## الأعمدة

| العمود | الوصف |
| --- | --- |
| `assignment_id` | رقم التكليف مثل `ASG-001` |
| `title` | عنوان المهمة |
| `owner` | اسم المسؤول |
| `email` | بريد المسؤول |
| `priority` | عاجل، مرتفع، متوسط، منخفض |
| `due_date` | تاريخ الاستحقاق |
| `status` | مكلفة، قيد التنفيذ، مكتملة، متأخرة، ملغاة |
| `details` | تفاصيل التكليف والمخرجات |
| `drive_link` | رابط الملف أو المسار في Google Drive |
| `assigned_by` | الجهة المكلفة، غالباً PMO |
| `email_sent_at` | وقت إرسال الإيميل |
| `created_at` | وقت الإنشاء |
| `updated_at` | آخر تحديث |

## Script Properties إضافية

أضف في Apps Script:

| Key | Value |
| --- | --- |
| `PMO_EMAIL` | بريدك لاستلام نسخة من إيميلات التكليف |

## إرسال تكليف من الداشبورد

1. افتح الداشبورد.
2. ادخل إلى `توزيع المهام`.
3. عبئ عنوان المهمة، المسؤول، الإيميل، الاستحقاق، الأولوية، التفاصيل، والرابط.
4. اضغط `إرسال التكليف`.

يجب أن يكون رابط Google Apps Script محفوظاً في صفحة الإعدادات.

## إرسال تكليف عبر API

```bash
curl -X POST "WEB_APP_URL" \
  -H "Content-Type: text/plain;charset=utf-8" \
  -d '{
    "action":"task_assignment",
    "title":"تجهيز قائمة الملفات المطلوبة",
    "owner":"منذر",
    "email":"m.alansari@mayadeen.sa",
    "priority":"عاجل",
    "due_date":"2026-07-05",
    "details":"تحديث الشيت بروابط الملفات النهائية",
    "drive_link":"https://drive.google.com/drive/folders/16hfR2XEBB9l-5YQIn0QMbrCvVpeA6ABV",
    "assigned_by":"PMO"
  }'
```

## ملاحظات

- أول تشغيل سيطلب صلاحية إرسال البريد من حساب Google الذي نشر Apps Script.
- يفضل أن يكون الإرسال باسم حساب PMO أو حساب مشروع رسمي.
- لا تحفظ كلمات مرور أو Webhook داخل الداشبورد.
